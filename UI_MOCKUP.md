## Key UI Elements

### Navigation
- Consistent header across all pages
- Clear navigation between sections
- Professional color scheme (dark header, light content)

### Die Lines Management
- File upload with drag-and-drop ready interface
- Grid layout for die line cards
- Visual preview placeholders
- Metadata display (dimensions, file type)

### Sheets Configuration
- Form-based input for sheet properties
- Material dropdown (paper, cardboard, vinyl, plastic)
- Granular margin controls
- Grid display of created sheets

### Ganging/Optimization
- Multi-step workflow:
  1. Select sheet
  2. Choose die lines + quantities
  3. Configure options
  4. Run optimization
  5. View results
  6. Export

- Visual canvas showing:
  - Sheet boundaries
  - Margin guidelines (dashed)
  - Placed die lines (color-coded)
  - Numbering for reference

- Real-time metrics:
  - Placement count
  - Utilization percentage
  - Waste percentage
  - Area calculations

- Export actions:
  - PDF generation
  - CSV report download

## Color Scheme
- Header: #2c3e50 (Dark blue-gray)
- Primary Action: #42b983 (Green)
- Secondary: #6c757d (Gray)
- Danger: #dc3545 (Red)
- Background: #f8f9fa (Light gray)
- Borders: #dee2e6 (Light border)

## Responsive Design
- Grid layouts adapt to screen size
- Mobile-friendly forms
- Touch-friendly buttons
- Scalable canvas

## User Experience
- Clear call-to-action buttons
- Helpful placeholder text
- Confirmation dialogs for destructive actions
- Loading states during async operations
- Error messages with clear guidance
